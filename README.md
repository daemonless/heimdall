## Deployment

### Podman Compose

```yaml
services:
  heimdall:
    image: ghcr.io/daemonless/heimdall:latest
    container_name: heimdall
    environment:
      - PUID=1000
      - PGID=1000
      - TZ=UTC
    volumes:
      - "/path/to/containers/heimdall:/config"
    ports:
      - 80:80
    restart: unless-stopped
```
